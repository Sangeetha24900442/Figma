## AIM:
To design, develop and deploy a web application for event registration.

## DESIGN STEPS:

### Step 1:
Create a new frame.

### Step 2:
Select any one preset size of your choice.

### Step 3:

### Step 4:
Import images as needed.

### Step 5:
Create pages based on your need and link them.

### Step 6:
